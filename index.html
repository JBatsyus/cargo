  1. Строительная техника и оборудование
    • Экскаваторы, бульдозеры, погрузчики.
    • Строительные машины (бетономешалки, краны, асфальтоукладчики).
    • Тяжёлое строительное оборудование (лифты, платформы).
  2. Сельскохозяйственная техника
    • Тракторы, комбайны, сеялки, косилки.
    • Оборудование для фермерских хозяйств (погрузчики, насосы, генераторы).
  3. Машины и автомобили
    • Грузовые автомобили, фургоны, автобусы.
    • Специализированные транспортные средства (автовышки, тягачи).
  4. Железнодорожное оборудование
    • Локомотивы, вагонные секции, платформы.
    • Запасные части и детали для железнодорожного транспорта.
  5. Тяжёлое промышленное оборудование
    • Генераторы, турбины, компрессоры.
    • Котлы, печи, агрегаты для переработки металлов.
    • Оборудование для нефтегазовой и горнодобывающей промышленности.
  6. Мебель и интерьерные изделия
    • Большие мебельные комплекты (например, офисная мебель, системы хранения).
    • Крупные элементы для гостиниц, ресторанов (столы, стойки, кресла).
  7. Производственное оборудование
    • Станки, пресс-формы, сварочные установки.
    • Линии для упаковки и переработки продукции.
  8. Деревянные и строительные материалы
    • Доски, балки, бетонные панели.
    • Строительные блоки, кирпичи, металлические конструкции.
  9. Тяжёлые запчасти и агрегаты
    • Двигатели, трансформаторы, трансмиссии.
    • Машинные части для различных отраслей промышленности.
  10. Морское и речное оборудование
    • Лодки, катера, яхты, судовые моторы.
    • Плавучие платформы, части для строительства кораблей.
  11. Металлоконструкции
    • Сварные конструкции, мосты, элементы каркасных зданий.
    • Стальные или алюминиевые конструкции для промышленных объектов.
  12. Техникум, лабораторное оборудование
    • Лабораторные установки, большие исследовательские приборы.
    • Оборудование для медицинских учреждений, включая диагностические устройства.
  13. Специализированные производственные линии
    • Линии для производства стекла, бетона, упаковки.
    • Оборудование для переработки отходов.
  14. Энергетическое оборудование
    • Трансформаторы, солнечные панели.
    • Газовые и угольные турбины, генераторные установки.
  15. Товары для крупных торговых и складских комплексов
    • Рекламные конструкции, баннеры, стеллажи.
    • Оборудование для хранения товаров, паллеты с крупными партиями товаров.
  16. Крупные и тяжёлые упаковки товаров
    • Большие партии промышленного оборудования в специальной упаковке.
    • Погрузочные контейнеры с товаром для оптовой продажи.
  17. Космическое и аэрокосмическое оборудование
    • Компоненты спутников, ракеты.
    • Часть материалов для строительства космических станций.
  18. Военное оборудование
    • Бронетехника, артиллерийские установки.
    • Части для военных судов, вертолётов, других вооружений.
  19. Культуры и природные ресурсы
    • Крупные партии сырья (нефть, уголь, металлы).
    • Продукция природных ресурсов, например, древесина, полезные ископаемые.
  20. Животные в крупных количествах
    • Перевозка большого числа сельскохозяйственных животных (коровы, лошади, овцы).
    • Экзотические животные, которые требуют особых условий транспортировки.
  21. Электрические и энергетические установки
    • Генераторные установки, распределительные панели.
